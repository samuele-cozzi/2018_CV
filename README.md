# My Resume (v.2018)

## Introduction

The objective of this project is to create a single webpage resume with react js, and use it as a template.

## Coding
### Prerequisites
- Git
- Node js
- bower (```npm install -g bower```)
- gulp (```npm install gulp-cli -g```)
- Firebase (```npm install -g firebase-tools```)

### Clone Repository and restore

```shell
$ git clone https://github.com/samuele-cozzi/2018_CV.git
$ cd .\2018_CV\
$ code .
```

```shell
$ bower install
$ npm install
```

### Build and Run

```shell
$ gulp build
$ gulp run
```

Alternatevly

```shell
$ gulp build
$ firebase login
$ firebase serve
```

### Deploy

```shell
$ firebase deploy
```
