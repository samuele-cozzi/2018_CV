## Coding
### Prerequisites
- Git
- Node js
- bower (```npm install -g bower```)
- gulp (```npm install gulp-cli -g```)
- Firebase (```npm install -g firebase-tools```)

### Clone Repository

```shell
$ git clone https://github.com/samuele-cozzi/2018_CV.git
$ cd .\2018_CV\
$ code .
```

### Build and Run

```shell
$ bower install
$ npm install
$ gulp build
$ gulp run
```

Alternatevly

```shell
$ gulp build
$ firebase login
$ firebase serve
```

### Deploy

```shell
$ firebase deploy
```
